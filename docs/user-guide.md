# AlphaJerry 用户指南

AlphaJerry 帮你从 A 股 5000 多只股票中，快速找到基本面靠谱的公司。

你不需要懂编程，也不需要会看财务报表。工具帮你算好了，你只需要看结论。

---

## 你能得到什么

每个月（或每个财报季之后），AlphaJerry 会给你三样东西：

**1. 一份 Top 20 推荐名单**

排在最前面的，就是全市场基本面最好的股票。每只股票都标好了类型：

| 类型 | 意思 | 举个例子 |
|------|------|---------|
| 🐎 千里马 | 正在高速成长的公司 | 营收增长 > 15%，处于扩张期 |
| 🛡️ 护城河 | 有品牌优势、毛利高的公司 | 竞争对手很难复制 |
| 🐮 现金牛 | 成熟稳赚的公司 | 利润稳定，常分红 |

**2. 一个评级**

每只股票都有一个评级，从好到差：

| 评级 | 意思 | 建议 |
|------|------|------|
| 👑 皇冠明珠 | 全市场顶尖，各方面都优秀 | 重点关注 |
| ⭐ 优秀白马 | 基本面扎实，稳当 | 可以长期关注 |
| 🔄 鸡肋/观察 | 有优点也有短板 | 再研究研究，不急 |
| ⚠️ 垃圾 | 基本面不好 | 暂时别碰 |
| 🚫 否决 | 有财务造假嫌疑或资不抵债 | 坚决回避 |

**3. 风险提醒**

如果某只股票的财务状况出现危险信号，AlphaJerry 会标记出来，并建议你加仓、持有、减仓还是止损。

## 怎么用

### 每个季度用一次

A 股每个季度发布一次财报（4 月、8 月、10 月底）。财报发布后，跑一次全流程，拿到最新的推荐名单。

### 每天可以做的事

如果你有持仓，可以每天看看：

- 持仓股的最新评分有没有变化
- 有没有触发风险警报
- 市场热点有没有影响到你持有的行业

### 查看某只股票

选中一只股票后，按这个顺序判断：

```
① 有没有被否决？                → 有的话直接放弃
② 财务上有没有风险？            → 负债率太高的话小心
③ 收入和利润在增长吗？          → 增长才是好公司
④ 股东回报怎么样？              → ROE 高才是好生意
```

什么情况会被否决？三种：一是净资产为负（资不抵债），二是经营现金流远低于净利润（造假嫌疑），三是关键数据缺失太多。触发任一条，这只股票就不参与评分和排名了。

## 几个提醒

1. **数据不是实时的** — AlphaJerry 看的是财报数据，不是股价。季报发布后才有最新数据
2. **只看基本面** — 不看 K 线、不看资金流向、不看政策消息。适合做长线参考
3. **不是炒股建议** — 评分反映的是财务健康度，不保证股价涨。投资决策需要你自己做
4. **可能有遗漏** — 好公司不一定分数高，有些优质公司可能因为数据不全被漏掉。建议把 Top 20 当线索，不是全部

## 常见问题

**需要我懂财务报表吗？**

不用。工具帮你算好了，你看评级和类型就行。

**多久用一次合适？**

每个季度财报后跑一次就够了。平时看看持仓变化。

**为什么有些股票评分很低？**

可能是增速慢、负债高、或者回报低。也不一定就是坏公司，公用事业类的公司本来就增速慢。

**评分高的股票一定会涨吗？**

不会。基本面好和股价涨是两回事。评分高的公司抗风险能力强，但短期股价受很多因素影响。
