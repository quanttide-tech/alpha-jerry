## 数据采集

### 数据源接口

| 接口 | 数据源 | 用途 |
|:--|:--|:--|
| `ak.stock_info_a_code_name()` | 东财 | 全部 A 股代码 + 名称（批量 1 次） |
| `ak.stock_financial_debt_new_ths(symbol, "按报告期")` | 同花顺 | 资产负债表 |
| `ak.stock_financial_benefit_new_ths(symbol, "按报告期")` | 同花顺 | 利润表（含同比增速） |
| `ak.stock_financial_cash_new_ths(symbol, "按报告期")` | 同花顺 | 现金流量表 |
| `ak.stock_profile_cninfo(symbol)` | 巨潮 | 个股概况：上市日期 |
| `sw_index_first_info()` + `index_component_sw(code)` | 申万指数 | 全 A 股 → 申万一级行业映射（缓存至 `data/缓存/sw_mapping.json`） |

### 空值策略

| 类别 | 字段 | 处理 |
|------|------|------|
| 必填 | Code / Name / Rev / NP / TA / TL / SE / CA / CL / OCF / PubDate / Period / ListDate / 行业属性 | 缺失则丢弃整行 |
| 补零 | InvP / NRI / LTL / FA / IA / ICF / CFF | 空值填 0 |
| 保留 | GP / OP / RE / APIC / EPS / BVPS / ROE / 增速字段 | 可空保留 |

### 异常值处理

| 异常 | 判定 | 处理 |
|:----|:-----|:-----|
| TA ≤ 0 | 理论不可能 | 丢弃 |
| SE ≤ 0 | 资不抵债 | 丢弃，标注 ST |
| 同比增速 > 500% | 并购等非经营因素 | Winsorize 到 500% |
| 比率 > 1000% | 分母极小失真 | Winsorize 到 ±1000% |

`数据质量` 列记录缺失衍生指标数量，超过 1/3 则不参与排名。

## 个股评分

### 流程

```
读取 data/财务/YYMMDD.xlsx
  → 一票否决筛选
  → 行业分类映射（申万 → 5 大类）
  → 成长性评分（营收增速 + 净利增速）
  → 稳健性评分（资产负债率 + 流动比率）
  → 资金回报评分（ROE）
  → 读取行业权重表
  → 综合分 = 成长分×w1 + 稳健分×w2 + 回报分×w3
  → 保存 data/财务/YYMMDD-评分.xlsx
```

### 一票否决

| 否决项 | 判定条件 | 实现方式 |
|--------|---------|---------|
| 造假嫌疑 | 经营现金流连续多年远低于净利润（OCF/NP < 0.5） | 规则自动 |
| 诚信问题 | 频繁更换审计机构 / 大股东质押率 > 70% / 信披违规 | LLM 联网搜索（二期） |
| 行业毁灭 | 技术路线被颠覆 | LLM 联网搜索（二期） |

否决股票写入 `否决` 列，不参与评分。

### 成长性得分

| 分数 | 营收增速 | 净利增速 |
|------|---------|---------|
| 9-10 | > 40% | > 50% |
| 7-8 | 25-40% | 30-50% |
| 5-6 | 15-25% | 15-30% |
| 3-4 | 5-15% | 0-15% |
| 1-2 | < 5%或负 | 负增长 |

取两项均值。缺失项不计入，分母自动减 1。

### 稳健性得分

| 分数 | 资产负债率 | 流动比率 |
|------|-----------|---------|
| 9-10 | < 40% | > 2.5 |
| 7-8 | 40-60% | 1.5-2.5 |
| 5-6 | 60-75% | 1.0-1.5 |
| 3-4 | 75-85% | 0.5-1.0 |
| 1-2 | > 85% | < 0.5 |

取两项均值。缺失不计入。

### 资金回报得分

| 分数 | ROE |
|------|-----|
| 9-10 | > 20% |
| 7-8 | 15-20% |
| 5-6 | 10-15% |
| 3-4 | 6-10% |
| 1-2 | < 6% |

### 行业权重

| 行业 | 成长性 | 稳健性 | 资金回报 |
|------|--------|--------|---------|
| 周期资源 | 0.25 | 0.35 | 0.40 |
| 大消费 | 0.30 | 0.30 | 0.40 |
| 证券金融 | 0.30 | 0.30 | 0.40 |
| 科技/制造 | 0.50 | 0.20 | 0.30 |
| 公用事业/基建 | 0.20 | 0.40 | 0.40 |

### 综合分

综合分 = 成长分 × 成长权重 + 稳健分 × 稳健权重 + 回报分 × 回报权重

## 个股评级

| 综合分 | 评级 |
|--------|------|
| 8.5 - 10.0 | 👑 皇冠明珠 |
| 7.0 - 8.4 | ⭐ 优秀白马 |
| 5.5 - 6.9 | 🔄 鸡肋/观察 |
| < 5.5 | ⚠️ 垃圾时间 |

否决股票标为「否决」，不参与评级。输出 `data/财务/YYMMDD-评级.xlsx`。

## 报告输出

取前 20 名，按综合分降序排列。

### 输出字段

| 字段 | 说明 |
|------|------|
| 股票代码 / 股票名称 | - |
| 公司类型 | 千里马 / 现金牛 / 护城河 |
| 行业分类 | 周期资源 / 大消费 / 证券金融 / 科技/制造 / 公用事业/基建 |
| 成长性 / 稳健性 / 回报性 | 各维度评分（0-10） |
| 综合分 | 加权总分（0-10） |
| 评级 | 皇冠明珠 / 优秀白马 / 鸡肋/观察 / 垃圾时间 |
| 核心亮点 | AI 生成，≤30 字 |
| 点评 | AI 点评，≤50 字 |

### 公司类型判定

| 条件 | 类型 |
|------|------|
| 科技/制造 且 营收增速 > 15% | 🐎 千里马 |
| 公用事业/基建 且 营收增速 > 15% | 🐎 千里马 |
| 周期资源 且 营收增速 > 30% | 🐎 千里马 |
| 大消费 且 毛利率 > 50% | 🛡️ 护城河 |
| 毛利率 > 40% 且 ROE > 15 | 🛡️ 护城河 |
| 其余 | 🐮 现金牛 |

### 行业分类确认

| 行业 | 常态分类 | 特殊情况 |
|------|---------|---------|
| 周期资源 | 现金牛 | 景气爆发期可为千里马 |
| 大消费 | 护城河 | 大众消费品可为千里马 |
| 证券金融 | 护城河（头部） | 特色券商可为千里马 |
| 科技/制造 | 千里马 | 平台型龙头可为护城河 |
| 公用事业/基建 | 现金牛 | 扩张期可为千里马 |

输出 `data/分析/YYMMDD-荐股.xlsx`。
